# AI-Assisted Credit Risk Platform

An end-to-end AI-powered credit risk assessment platform that combines:

- Document AI
- Credit Risk Modeling
- Fraud Detection
- Rule Engine
- Explainable AI
- LLM Underwriter Reports

## Tech Stack

- Python 3.12
- Streamlit
- scikit-learn + XGBoost
- SHAP
- PyMuPDF + EasyOCR
- Gemini Flash
- GPT-4.1 mini
- ReportLab

## Workflow

Loan PDF + Bank Statement PDF

-> Document Extraction

-> Feature Engineering

-> Risk Prediction

-> Fraud Detection

-> SHAP Explainability

-> LLM Underwriter Report

-> PDF Export

-> Human Decision

## Guardrail

The LLM never approves or rejects loans. It only summarizes evidence for a human underwriter.
